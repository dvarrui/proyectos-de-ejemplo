# Estructura condicional

Ejemplo 1:

```
edad = gets
if (edad > “18”)
  puts “Eres mayor de edad”
end
```

---

Ejemplo 2:

```
edad = gets.to_i
if edad > 18
  puts “Eres mayor de edad”
end
```

---

Ejemplo 3:

```
edad = gets.to_i
puts “Eres mayor” if (edad > 18)
```
